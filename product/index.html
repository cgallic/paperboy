Before you connect anything

Reasonable questions, answered directly.

Does Paperboy need every repository?

No. GitHub is an optional source lane. If you use it, select one to five public repositories. This local preview uses fictional fixtures only.

Can Paperboy change my code?

No. The planned GitHub access is read-only. The product does not write code, issues, pull requests, or run autonomous actions.

Does Paperboy read my inbox?

No. Paperboy gives you a forwarding alias. It sees only messages you intentionally forward to that address. Gmail OAuth is excluded.

What arrives by email?

A capped edition: Today in 60 Seconds, Repo Radar, Research Worth Reading, and Watchlist / Do Nothing. Every substantive item links to evidence.

Is this the same as the open-source project?

The hosted concept builds on the project's scoring and digest ideas. The free self-hosted SQLite/systemd/Discord project remains available and separate.
